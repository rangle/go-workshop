# Go Workshop

# Installation

- Install Go: https://golang.org/doc/install
- _(optional) [Install VS Code extension](https://marketplace.visualstudio.com/items?itemName=golang.go)_
- Verify installation `go version`
- compile `example1`
  ```Bash
  cd example1
  go build
  ./hello
  ```
- compile `example2`
  ```Bash
  cd example2
  go build
  ./server
  open http://localhost:8080
  ```
- compile `example3`
  ```Bash
  cd example3
  go build
  ./createimage
  ```
- compile `example4`
  ```Bash
  cd example4
  go build
  ./concurrency
  ```
